# Pre-Advising System

## Overview

This repository contains a web-based Pre-Advising system designed to assist students and advisors in planning and managing academic courses. The system allows users to search for curricula, view course details, and plan their academic path.

## Features

*   Searchable curriculum database
*   Course details viewing
*   Academic planning and advising tools
*   User-friendly interface

## Technologies Used

*   HTML
*   CSS
*   JavaScript
*   Firebase (for database and authentication)

## Installation

To run the application locally, follow these steps:

1.  Clone the repository using `git clone https://github.com/your-username/pre-advising.git`
2.  Install the required dependencies using `npm install`
3.  Start the application using `npm start`
4.  Open your web browser and navigate to `http://localhost:3000`

## Usage

1.  Search for curricula using the search bar
2.  View course details by clicking on the course name
3.  Plan your academic path using the advising tools
